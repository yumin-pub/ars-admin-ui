# Input

사용자로부터 텍스트를 입력받는 기본 입력 컴포넌트입니다.

## Overview

지원 기능

- 3가지 Variant
  - Outline
  - Underline
  - Ghost
- 2가지 Size
  - Medium
  - Large
- Label 지원
- Helper Message 지원
- Prefix / Suffix 지원
- Clear Button 지원
- Success / Error 상태 지원
- Disabled / Readonly 지원
