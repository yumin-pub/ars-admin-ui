# Table

Table 컴포넌트는 공통 테이블 UI를 제공합니다.

- 컬럼(Column)과 데이터(Data)를 분리하여 관리합니다.
- 제네릭(Generic)을 사용하여 행(Row) 타입을 안전하게 지정할 수 있습니다.
- `render`를 통해 특정 셀을 원하는 형태로 렌더링할 수 있습니다.
- Loading, Empty 상태를 기본으로 제공합니다.
