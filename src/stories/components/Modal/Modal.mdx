# Modal

사용자에게 중요한 정보를 전달하거나 입력 및 확인 작업을 처리하기 위한 공통 모달 컴포넌트입니다.

Header, Body, Footer 영역을 조합하여 다양한 형태의 모달을 구성할 수 있습니다.

---

## Import

```tsx
import { Modal } from '@/components/Modal';
```

---

## Playground

## Included Examples

### Alert Modal

간단한 처리 결과나 알림 메시지를 전달하는 모달입니다.

확인 버튼 또는 X 버튼을 누르면 닫힙니다.

### Confirm Modal

사용자의 확인이 필요한 작업에 사용하는 모달입니다.

취소 버튼과 X 버튼은 모달을 닫고, 확인 버튼은 작업 처리 후 모달을 닫습니다.

### Form Modal

Input, Select, Checkbox를 조합하여 데이터를 입력받는 모달입니다.

취소 버튼은 모달을 닫고, 저장 버튼은 저장 처리 후 모달을 닫습니다.

### Body Actions

Footer를 사용하지 않고 Body 내부에 액션 버튼을 배치한 예시입니다.

### No Footer

Footer 없이 Header와 Body 영역만 사용하는 모달입니다.
