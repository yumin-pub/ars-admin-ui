# ARS Admin UI System

React와 TypeScript를 기반으로 구축한 관리자 UI 시스템이다.

HTML과 SCSS로 구성된 Core UI를 먼저 설계하고, React 컴포넌트는 Core UI를 편리하게 사용할 수 있도록 Wrapper 형태로 구현한다.

## 프로젝트 목표

- 다양한 관리자 화면에서 재사용할 수 있는 공통 UI 컴포넌트를 구축한다.
- 디자인 토큰과 일관된 클래스 규칙을 기반으로 스타일을 관리한다.
- Storybook을 통해 컴포넌트의 사용 방법과 상태를 문서화한다.
- 포트폴리오 제출용 v1을 완성한 뒤 실무형 디자인 시스템으로 지속적으로 확장한다.

## 설계 원칙

- 특정 페이지보다 공통 컴포넌트를 우선한다.
- HTML, Style, Logic, Data의 역할을 분리한다.
- 기존 컴포넌트와 Naming을 먼저 확인한다.
- 미완성 기능은 v1에 억지로 포함하지 않고 Roadmap으로 관리한다.

## 현재 컴포넌트

- Button
- Input
- Radio
- Checkbox
- Select
- Modal
- Table
- Pagination
