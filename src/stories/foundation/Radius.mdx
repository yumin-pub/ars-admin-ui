# Radius

Radius Token은 컴포넌트의 모서리 곡률을 일관된 기준으로 관리하기 위한 디자인 토큰이다.

Button, Input, Card, Modal 등 컴포넌트 특성에 따라 동일한 Radius를 사용하여 UI의 통일성을 유지한다.

## Why Radius Tokens?

Radius는 프로젝트의 시각적 인상을 결정하는 중요한 요소이다.

공통 토큰으로 관리하면 모든 컴포넌트가 동일한 기준을 공유할 수 있으며, 디자인 변경 시에도 일괄적으로 대응할 수 있다.

## Usage Guidelines

### Good

```scss
border-radius: $radius-md;
```

### Bad

```scss
border-radius: 7px;
```

컴포넌트에서 직접 Radius를 작성하지 않고 Radius Token을 사용한다.

## Summary

Radius Token을 공통 기준으로 사용하여 프로젝트 전반의 시각적 일관성과 재사용성을 확보한다.
