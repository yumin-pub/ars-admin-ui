# Spacing

Spacing Token은 UI의 여백을 일관된 기준으로 관리하기 위한 디자인 토큰이다.

Margin, Padding, Gap을 동일한 Scale로 관리하여 컴포넌트 간 간격과 화면의 리듬을 유지한다.

## Why Spacing Tokens?

Spacing은 화면의 완성도를 결정하는 가장 중요한 요소 중 하나이다.

프로젝트마다 임의의 여백을 사용하는 대신 공통 Scale을 정의하여 일관성을 유지한다.

Spacing Token을 사용하면 새로운 프로젝트에서도 동일한 기준을 그대로 사용할 수 있으며, 디자인 변경 시에도 전체 여백 체계를 쉽게 관리할 수 있다.

## Usage Guidelines

Spacing은 Margin, Padding, Gap에만 사용한다.

### Good

```scss
padding: $space-4;
margin-bottom: $space-6;
gap: $space-2;
```

### Bad

```scss
padding: 17px;
margin-bottom: 23px;
gap: 7px;
```

가능한 경우 기존 Scale을 우선 사용한다.

새로운 값을 추가하기 전에 기존 Token으로 해결 가능한지 먼저 확인한다.
