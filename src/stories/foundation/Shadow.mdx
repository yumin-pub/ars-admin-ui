# Shadow

Shadow Token은 UI 요소의 깊이와 계층을 시각적으로 표현하기 위한 디자인 토큰이다.

Dropdown, Modal, Drawer, Toast처럼 화면 위에 떠 있는 요소의 층위를 일관된 기준으로 구분한다.

## Why Shadow Tokens?

Shadow 값을 컴포넌트마다 직접 작성하면 비슷하지만 서로 다른 그림자가 반복해서 사용될 수 있다.

공통 Shadow Token을 사용하면 요소의 높이와 강조 수준을 동일한 기준으로 유지할 수 있으며, 디자인 변경 시에도 토큰만 수정하여 전체 UI에 반영할 수 있다.

## Token Scale

| Token | Usage |
| --- | --- |
| `$shadow-none` | 그림자가 필요하지 않은 기본 상태에 사용한다. |
| `$shadow-sm` | Input Focus, Button Hover 등 낮은 수준의 강조에 사용한다. |
| `$shadow-md` | Dropdown, Select Option 등 떠 있는 요소에 사용한다. |
| `$shadow-lg` | Modal, Drawer 등 높은 층위의 레이어에 사용한다. |
| `$shadow-xl` | Toast, Floating Layer 등 가장 높은 강조에 사용한다. |

## Usage Guidelines

컴포넌트의 시각적 층위에 맞는 Shadow Token을 선택한다.

### Good

```scss
.ui-dropdown {
    box-shadow: $shadow-md;
}

.ui-modal__panel {
    box-shadow: $shadow-lg;
}
```

### Bad

```scss
.ui-dropdown {
    box-shadow: 0 3px 7px rgba(0, 0, 0, 0.09);
}

.ui-modal__panel {
    box-shadow: 0 10px 18px rgba(0, 0, 0, 0.13);
}
```

새로운 그림자를 추가하기 전에 기존 Shadow Scale로 표현할 수 있는지 먼저 확인한다.

## Source

Shadow Token은 다음 파일에서 관리한다.

```txt
src/styles/tokens/_shadow.scss
```

```scss
$shadow-none: none;

$shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);

$shadow-md: 0 4px 8px rgba(0, 0, 0, 0.08);

$shadow-lg: 0 8px 16px rgba(0, 0, 0, 0.12);

$shadow-xl: 0 12px 24px rgba(0, 0, 0, 0.16);
```

## Summary

Shadow Token을 통해 UI 요소의 깊이와 계층을 공통 기준으로 관리한다.

이를 통해 레이어 간 관계를 명확하게 표현하고, 여러 프로젝트에서도 동일한 Shadow 체계를 재사용할 수 있다.
